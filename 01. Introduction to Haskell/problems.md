## Задача 01

```hs
ghci> sphereVolume 4
268.082573106329
```

## Задача 02

Да се дефинира функция, която приема коефициентите на полином от втора степен
и връща дали той има реални корени.

```hs
ghci> hasRealRoots 1 2 1
True
ghci> hasRealRoots 1 0 1
False
```

```hs
ghci> distance (1,2) (7,3)
6.082762530298219
```

## Задача 04

```hs
ghci> max3 5 (-1) 3

```

## Задача 05

Да се дефинира функция, която проверява дали дадено цяло число е нула.
Напишете функцията по поне два различни начина.

```hs
ghci> isZero 0
True
ghci> isZero 4
False
```

## Задача 06

Да се дефинират операторите `&&&` и `|||`, които са съответно конюнкция
и дизюнкция на булеви стойности.

```hs
ghci> False &&& True
False
ghci> False ||| True
True
```

## Задача 08

Да се дефинира функция, която приема цяло число, представящо секунди,
и връща наредена тройка от часовете, минутите и секундите,
на които са равни секундите, които се подават като параметър.

```hs
ghci> hms 3661
(1,1,1) -- 1 час, 1 минута и 1 секунда
```

## Задача 09

Да се дефинират следните функции, работещи над рационални числа.
Рационални числа ще представяме като наредена двойка от две цели числа,
представящи съответно числителя и знаменателя на рационалното число:

- `mkRat :: Int -> Int -> Rational`, която създава рационално число по подадени
  негов числител и знаменател. Кога едно такова число е невалидно?
- `simplify :: Rational -> Rational`, която опростява дробта, чрез която е представено
  даденото рационално число. Например, числото $\frac{2}{4}$ се опростява до $\frac{1}{2}$;
- `add :: Rational -> Rational -> Rational`, която добавя две рационални числа, като ги опростява.

## Задача 11

Да се дефинира функция, която намира сумата от цифрите на някакво цяло число.
Ще искаме тази сума винаги да бъде положителна, независимо от знака на числото.

```hs
ghci> sumDigits 12340
10
```
